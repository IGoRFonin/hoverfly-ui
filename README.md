# Hoverfly-ui

## Install dependencies
```bash
$ yarn install
```

## Run lint
```bash
$ yarn lint
```

## Development
Before run project need to start any hoverfly server.

### Run hoverfly server
```bash
$ hoverfly -webserver -cors -dev

# or run with auth

$ hoverlfy -webserver -cors -dev -auth -username admin -password admin
```

### Run project
```bash
$ yarn start
```
